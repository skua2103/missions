# FRONTLINE 仕様書

## 1. 画面一覧

| 画面名 | 主な機能 |
|--------|----------|
| ホーム画面 | 各機能へのナビゲーション |
| マスタータスク設定画面 | 長期目標タスクの作成・編集・削除 |
| デイリータスク設定画面 | 今日実行するタスクの選択と時間設定 |
| タスク実行画面 | タイマー機能、経験値取得、達成アニメーション |
| ユーザープロフィール画面 | 進捗状況、経験値、称号、継続日数の表示 |
| ランキング画面 | ユーザー間の比較（経験値ランキング等） |

## 2. 画面遷移フロー

### 2-1. 新規ユーザーの初回フロー

```
ユーザー登録/ログイン
  ↓
マスタータスク設定画面（初回誘導）
  ├─ タスク設定する → ホーム画面へ
  └─ 「ホームへ」ボタン → ホーム画面へ（設定スキップ可能）
  ↓
ホーム画面
  ↓
デイリータスク設定画面
  ↓
タスク実行画面
```

### 2-2. 既存ユーザーの日常フロー

```
ログイン
  ↓
ホーム画面
  ↓
デイリータスク設定画面（今日のタスクを選ぶ）
  ↓
タスク実行画面
  ↓
完了後 → ユーザープロフィール（進捗確認）
```

### 2-3. モチベーション確認フロー

```
ホーム画面
  ├─ ランキング画面（他ユーザーとの比較）
  └─ ユーザープロフィール画面（自分の成長確認）
```

## 3. データモデル

### 3-1. UserProfile（ユーザープロフィール）

| フィールド名 | 型 | 説明 |
|-------------|-----|------|
| user | OneToOneField | Djangoのユーザーモデルと紐付け |
| total_exp | Integer | 累計経験値 |
| rank | CharField | 称号（Iron, Bronze, Silver等） |
| current_streak | Integer | 現在の継続日数 |
| max_streak | Integer | 最高継続日数記録 |

### 3-2. MasterTask（マスタータスク）

| フィールド名 | 型 | 説明 |
|-------------|-----|------|
| user | ForeignKey | UserProfileと紐付け |
| task_name | CharField | タスク名（例: プログラミング学習） |
| target_hours | Integer | 最終目標時間（分単位） |
| completed_hours | Integer | 現在の達成時間（分単位） |
| created_at | DateTimeField | 作成日時 |

### 3-3. DailyMission（デイリーミッション）

| フィールド名 | 型 | 説明 |
|-------------|-----|------|
| user | ForeignKey | UserProfileと紐付け |
| master_task | ForeignKey | MasterTaskから選択 |
| target_minutes | Integer | その日の目標時間（分） |
| completed_minutes | Integer | 実行した時間（分） |
| date | DateField | ミッションの日付 |
| is_completed | Boolean | 完了フラグ |

## 4. 主要機能

### 4-1. 経験値システム

- **計算式:** 実行時間（分） × 10 = 獲得経験値
- **例:** 30分タスク実行 → 300 EXP獲得

### 4-2. 称号（ランク）システム

| 称号 | 必要経験値 |
|------|-----------|
| Iron | 0 - 999 |
| Bronze | 1,000 - 4,999 |
| Silver | 5,000 - 9,999 |
| Gold | 10,000 - 19,999 |
| Platinum | 20,000 - 49,999 |
| Diamond | 50,000+ |

### 4-3. 継続日数（ストリーク）

- デイリーミッションを1つでも完了した日をカウント
- 1日でも途切れたらリセット
- 最高記録は保持

### 4-4. タイマー機能

- リアルタイム経験値表示
- プログレスバー表示
- 完了時の祝福アニメーション（紙吹雪エフェクト）
